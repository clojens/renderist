Renderist

Generate images hosted in the cloud from textual representation stored in github.

Usage

  1. Write your diagram using one of supported syntax
  2. Save it as a gist
  3. Directly access your image via a dedicated URL 
    http://renderist.herokuapp.com/GIST/FILE.png
    (where GIST is your gist id and FILE your gist file name. Both svg and png format are available.)
