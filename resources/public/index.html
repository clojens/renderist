Renderist

Generate images hosted in the cloud from textual representation stored in github.

Usage

  1. Write your diagram using one of supported syntax
  2. Save it as a gist: its URL will be similar to https://gist.github.com/3473417 and can contain several files, like sequence.txt
  3. Directly access your image via a dedicated URL: http://renderist.herokuapp.com/3473417/sequence.png
